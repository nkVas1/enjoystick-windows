# Icon placeholder

Place `enjoystick.ico` in this directory and uncomment the `IDI_APP ICON` line
in `enjoystick.rc`, then rebuild.

Recommended ICO sizes: **16×16, 32×32, 48×48, 256×256** (32-bit RGBA).

The design system palette for the icon:
- Background: `#0C0C12` (SurfaceBase obsidian)
- Primary shape: `#E8C97A` (GoldHi)
- Accent highlight: `#F5DFA0` (GoldBright)
